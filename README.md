# Prediciendo reclamos de seguro vehicular
Este proyecto muestra el uso de regresión logística para predecir si los clientes de una empresa de seguros vehiculares harás un reclamo de su póliza de seguro. Más específicamente, aquí comparo diferentes modelos de regresión para hallar cuál característica individual es el mejor predictor del resultado deseado, utilizando métricas como la exactitud y el área bajo la curva ROC. El archivo `car_insurance.csv` contiene el conjunto de datos utilizado, y el cuaderno de Jupyter `insurance_claim_prediction.ipynb` muestra el proceso de análisis y los resultados.
___
# Predicting car insurance claims
This project shows the use of logistic regression to predict whether clientes of a car insurance company will make a claim on their insurance. More specifically, I compare several regression models to find out which single feature is the best predictor of the desired outcome, using metrics such as accuracy and the ROC AUC score. The file `car_insurance.csv` contains the dataset used, and the notebook `insurance_claim_prediction.ipynb` shows the analysis and results.

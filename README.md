# Prediciendo reclamos de seguro vehicular
Este proyecto muestra el uso de regresión logística para predecir si los clientes de una empresa de seguros vehiculares hará un reclamo de su póliza de seguro. Más específicamente, aquí comparo diferentes modelos de regreción para hallar cuál característica individual es el mejor predictor del resultado deseado, utilizando métricas como la exactitud y el área bajo la curva ROC.

# Predicting car insurance claims
This project shows the use of logistic regression to predict whether clientes of a car insurance company will make a claim on their insurance. More specifically, I compare several regression models to find out which single feature is the best predictor of the desired outcome, using metrics such as accuracy and the ROC AUC score.
